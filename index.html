🚀 如何将本工具部署到网上并获得链接？
1️⃣ GitHub Pages：将此文件保存为 index.html，新建 GitHub 仓库上传，进入 Settings → Pages → 选择 main 分支，即可获得 https://你的用户名.github.io/仓库名/ 链接。
2️⃣ Netlify Drop：访问 app.netlify.com/drop，直接把本文件拖进去，一秒生成公开链接。
3️⃣ Vercel / Cloudflare Pages 同样简单。数据完全存储在浏览器本地，换设备前记得“导出备份”即可迁移。
💡 或者直接双击 HTML 文件在本地使用，所有数据自动保存。
⚡ 所有提示词均保存在你浏览器的 localStorage 中，关闭页面也不会丢失。支持标签组合筛选、全文搜索。
